# Guide déploiement VPS (MT5)

1. Utiliser un VPS faible latence proche du broker.
2. Installer MT5 + copier EA et presets.
3. Activer l'AutoTrading et vérifier le smiley EA.
4. Désactiver la mise en veille du VPS.
5. Mettre en place une routine hebdomadaire:
   - contrôle drawdown,
   - vérification des logs,
   - contrôle des symboles actifs.
6. Prévoir redondance: snapshot système + sauvegarde presets.
